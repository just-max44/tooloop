# Store Metadata — Tooloop (MVP)

## Positionnement

Tooloop est une application locale de prêt d’objets entre voisins pour réduire les achats inutiles et renforcer la confiance de quartier.

## Catégorie suggérée

- Google Play: Lifestyle
- App Store: Lifestyle

## Description courte (Play Store)

Prêtez et empruntez des objets près de chez vous, simplement et en confiance.

## Description longue (Play Store / App Store)

Tooloop vous aide à partager les objets du quotidien entre voisins.

- Publiez un objet en quelques secondes
- Trouvez ce dont vous avez besoin autour de vous
- Envoyez une demande de prêt en quelques clics
- Consultez les profils, badges et avis pour échanger en confiance

Notre mission: réduire le gaspillage, encourager la réutilisation, et créer une communauté locale plus solidaire.
