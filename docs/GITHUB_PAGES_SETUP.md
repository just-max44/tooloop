# GitHub Pages — publication des pages légales

## 1) Pousser le repo sur GitHub

Assure-toi que ce repo est sur GitHub (branche `main`).

## 2) Activer GitHub Pages

Dans ton repo GitHub :

- `Settings` → `Pages`
- `Build and deployment` → `Source`: **Deploy from a branch**
- `Branch`: **main**
- `Folder`: **/docs**
- `Save`

## 3) URLs légales obtenues

Si ton repo est `OWNER/REPO`, les URLs seront :

- Politique de confidentialité: `https://OWNER.github.io/REPO/privacy.html`
- CGU: `https://OWNER.github.io/REPO/terms.html`

## 4) Où les mettre

- Google Play Console → App content → Privacy policy
- App Store Connect → App Privacy / App Information
- Fiche store: `docs/STORE_METADATA.md`

## 5) Vérification rapide

Ouvre ces 3 URLs :

- `https://OWNER.github.io/REPO/`
- `https://OWNER.github.io/REPO/privacy.html`
- `https://OWNER.github.io/REPO/terms.html`

Attends 1 à 5 minutes après activation si ça ne répond pas immédiatement.
